# ZSTronics_MERN_Task

## How To Run
Create an Atlas URI connection parameter in `ZSTronics_MERN_Task/server/.env` with your Atlas URI:


Start server:
```
cd ZSTronics_MERN_Task/server
npm install
npm start
```

Start Web server
```
cd ZSTronics_MERN_Task/client
npm install
npm start
```
